npm-license-analyzer

  • Enter npm package to list its dependencies together with the corresponding licenses.
  • Displayed licenses belong to the latest version of the dependency. If the package isn't using the latest version of the dependency, explicit check may be required.
  • All information is retrieved from the corresponding npm and github pages.
  • npm-license-analyzer is not affiliated with npm, Inc. in any way.
